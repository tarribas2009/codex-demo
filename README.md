# Hola App

Aplicacion web estatica muy simple con un boton que muestra el mensaje `Hola`.

## Abrir en local

1. Abre una terminal en `G:\Mi unidad\Codex\Proyecto 1`
2. Ejecuta `python -m http.server 8000`
3. Abre [http://localhost:8000](http://localhost:8000)

## Archivos principales

- `index.html`: estructura de la pagina
- `styles.css`: estilos visuales
- `src/main.js`: comportamiento del boton
